> Vue.js Meta Framework to create complex, fast & universal web applications *quickly*.

## Installation

```bash
$ npm install
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# generate static project
$ npm run start:generate
```
